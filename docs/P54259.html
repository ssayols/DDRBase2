ATN1

Atrophin-1 (Dentatorubral-pallidoluysian atrophy protein)

ATN1 P54259

Gene names

ATN1 D12S755E DRPLA

Function

FUNCTION: Transcriptional corepressor. Recruits NR2E1 to repress transcription. Promotes vascular smooth cell (VSMC) migration and orientation (By similarity). Corepressor of MTG8 transcriptional repression. Has some intrinsic repression activity which is independent of the number of poly-Gln (polyQ) repeats. {ECO:0000250|UniProtKB:O35126, ECO:0000269|PubMed:10085113, ECO:0000269|PubMed:10973986}.
